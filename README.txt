Для запуска в Code::Blocks IDE с правильной кодировкой:
1. В Project->Properties->Project settings->Project's build options->Other compiler options

Если 1. не достаточно:
2.Settings->Editor->Encoding settings->Use encoding when opening files: UTF-8
